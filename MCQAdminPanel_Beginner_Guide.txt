MCQAdminPanel Beginner Guide
============================

Audience
--------

This guide is for someone using MCQAdminPanel for the first time.

You do not need backend knowledge. You only need the app and your question data.

1) What This Project Does
-------------------------

MCQAdminPanel lets you:

1. Create multiple-choice questions in an admin workspace.
2. Edit, organize, reorder, and tag those questions.
3. Import and export question banks as JSON.
4. Export answer keys.
5. Build a standalone `exam.html` file.

The app you see in this repository is the shared web app layer of the project.

2) What Is In This Repository
-----------------------------

This repository contains the shared browser app files directly at the root:

1. `index.html`
2. `css/`
3. `js/`
4. `icons/`
5. `vendor/`
6. `standalone/`

In the full local project, this shared app is also packaged into:

1. A Windows Electron app
2. An Android Capacitor app

3) How To Start
---------------

Browser version:

1. Open the repo with a local static server such as Live Server.
2. Open `index.html`.
3. Start working with questions.

4) Main Parts Of The App
------------------------

1. Workspace Tools
   - Add Question
   - Undo / Redo
   - Delete actions
   - Bulk tools
   - Snapshots

2. Question Cards
   - Shows saved questions
   - Supports selection
   - Supports reorder
   - Lets you open the editor

3. Question Editor
   - Edit question text
   - Edit options
   - Mark the correct answer
   - Set topic, difficulty, marks, tags, and explanation

4. JSON Tools
   - Import JSON
   - Export selected JSON
   - Export full JSON
   - Export answer key

5. Exam Settings
   - Configure and build `exam.html`

6. Utility Tools
   - Print Questions
   - Print Answer Key
   - Snapshot Library
   - Test Harness

5) First-Time Workflow
----------------------

Step 1: Add your first question

1. Click `Add Question`.
2. Type the question text.
3. Add at least 2 options.
4. Select the correct option.
5. Click `Save Question`.

Step 2: Add more questions

1. Repeat the same process.
2. Questions appear as cards in the main area.

Step 3: Organize your questions

1. Search, filter, or reorder cards.
2. Select multiple cards if you want bulk actions.

Step 4: Back up your data

1. Open `JSON`.
2. Export the full question bank as JSON.

Step 5: Build the exam file

1. Open `Exam`.
2. Adjust settings.
3. Build `exam.html`.

6) Question Rules
-----------------

A normal question needs:

1. Non-empty question text
2. Between 2 and 6 options
3. One valid correct answer

For JSON import, the app is flexible:

1. Question text is required
2. Option values are required
3. Missing metadata can be generated automatically

7) JSON Import / Export
-----------------------

Import JSON:

- Accepts an array of questions or `{ "questions": [...] }`
- Appends into the current workspace
- Generates missing metadata when possible
- Rejects invalid rows and reports them

Export JSON:

- Full export keeps current question order
- Selected export keeps selected question order
- Answer key export is also available

8) Snapshots
------------

Snapshots are local restore points inside the app.

Use them when:

- you are about to make a big change
- you want a quick rollback point

Use JSON export when:

- you want a real backup file
- you want to move the data elsewhere

9) Helpful Features
-------------------

MCQAdminPanel also includes:

- dark theme
- high contrast mode
- keyboard shortcuts
- command palette
- undo / redo
- duplicate detection
- math rendering with MathJax

10) Best Habits
---------------

1. Export JSON regularly.
2. Save a snapshot before bulk edits.
3. Keep topic and tag naming consistent.
4. Review the answer key before publishing.
5. Run the test harness before final delivery.
